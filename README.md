# Tally string times with reduce()

## Description

The aim of this exercise is to obtain the total number of hours, minutes and seconds of a list of videos using the reduce() method.

## Javascript Used

* querySelectorAll()
* data attribute
* map()
* split()
* parseFloat()
* reduce()
* Math.floor()
* %
